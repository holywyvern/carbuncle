Overview

An Array represents a list of items in sequence. To create new arrays usually you use it from literals.

Examples:

my_list = [1, 2, 3] # my_list is an Array of 3 elements
my_list[0] # => 1 Arrays start from 0.
my_list.first # => also 1.
